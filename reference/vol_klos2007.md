# Estimate tree volume in Manitoba using Kozak (1988) form with Klos (2004) parameters

Estimate tree volume in Manitoba using Kozak (1988) form with Klos
(2004) parameters

## Usage

``` r
vol_klos2007(DBH, height, species, subregion = "Province")
```

## Arguments

- DBH:

  Numeric vector. Diameter at breast height (cm).

- height:

  Numeric vector. Total height (m).

- species:

  Character vector. Species code (e.g., "PICE.MAR").

- subregion:

  Optional Ecozone or Ecoregion identifier

## References

Klos, R. J., Wang, G. G., Dang, Q.-L., & East, E. W. (2007). *Taper
equations for five major commercial tree species in Manitoba, Canada*.
Western Journal of Applied Forestry, 22(3), 163–170.

Kozak, A. (1988). A variable-exponent taper equation. *Canadian Journal
of Forest Research*, 18, 1363–1368.
