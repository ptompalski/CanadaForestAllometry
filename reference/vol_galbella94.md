# Estimate tree volume in Saskatchewan using Kozak (1988) form with Gal & Bella (1994) parameters

Estimate tree volume in Saskatchewan using Kozak (1988) form with Gal &
Bella (1994) parameters

## Usage

``` r
vol_galbella94(DBH, height, species)
```

## Arguments

- DBH:

  Numeric vector. Diameter at breast height (cm).

- height:

  Numeric vector. Total height (m).

- species:

  Character vector. Species code (e.g., "PICE.MAR").
